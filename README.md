### 项目介绍
后端使用SpringCloud Alibaba开发，移动端使用Flutter2.x构建，管理后台使用Vue 3.0 + Element Plus 进行构建，并在支付上接入数字货币（比特币、以太坊UDST）支付，后端采用Hadoop与Flink等大数据框架构建实时计算与离线计算体系。
